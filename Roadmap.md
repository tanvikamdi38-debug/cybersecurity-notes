# 🛡️ Cybersecurity Knowledge Base

Welcome to my Cybersecurity Knowledge Base!

This repository documents my structured learning journey across cybersecurity. It is designed as a step-by-step roadmap covering cybersecurity fundamentals, networking, Security Operations Center (SOC), incident response, threat intelligence, blue team operations, digital forensics, and advanced security topics.

The objective of this repository is to build a well-organized knowledge base through structured notes, hands-on learning, practical workflows, and continuous learning.

This repository is built from my learning through:

- HTB Academy

- VirusTotal investigations
